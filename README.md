## 🚀 Sobre mim

Sou dev focado em **transformar problema complexo em solução simples e eficiente**. Trabalho principalmente com **Java e Python**, sempre buscando código limpo, arquitetura MVC e automação de verdade. Fora do código, curto entender como redes e eletrônica funcionam por baixo do capô — por isso venho estudando Cisco e brincando com Arduino.

- 🎓 **Formação:** Técnico em Desenvolvimento de Sistemas (SENAI/SC)
- 👨‍💻 **Atual:** Curso de Sistemas na **CentroWEG**
- 🎯 **Objetivo:** Engenharia de Software e Análise de Dados
- 🌱 **Explorando:** Redes (Cisco) e eletrônica/IoT (Arduino)
- ⚡ **Fun fact:** se travar, provavelmente é `NullPointerException`
